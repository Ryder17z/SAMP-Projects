This folder contains lists of ALL MTA:RACE, SA-MP and Default GTA SA objects.

dir.bat      =  Generate a relative directory listing without support for subfolders
objects.txt  =  Generated list of paths leading to object lists; *.IDE or *.MODELS
gta          =  Object Data Folder containing Default GTA SA objects
mta          =  MTA Race object lists

obj_injected.txt and obj_update.txt holds vital information about the current state of the object lists conversion

SEE objects2.pwn for reference
